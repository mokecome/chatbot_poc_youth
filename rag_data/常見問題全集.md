## 二、各科室及中心業務職掌

青年事務局依組織規程設立"三科一室"，並下設社會企業中心，形成覆蓋青年發展全領域的業務體系：

### 1. 綜合規劃科

- **核心職能**：青年發展政策研訂與綜合規劃
- **重點工作**：
  1. 編制年度青年政策白皮書
  2. 推動青年多元學習與社團參與
  3. 開展青年設計與藝文發展培育
  4. 組織青年國際交流及體驗學習活動

### 2. 職涯發展科

- **核心職能**：青年就業創業支援與生涯發展引導
- **重點工作**：
  1. 提供青年創業輔導與就業見習服務
  2. 舉辦職涯規劃講座與培訓
  3. 開展創新創業競賽等能力提升活動
  4. 推動青年職場體驗專案

### 3. 公共參與科

- **核心職能**：青年社會參與能力培育與實踐推動
- **重點工作**：
  1. 培育青年公共議題參與意識與能力
  2. 組織志工培訓及志願服務計劃
  3. 輔導青年成立及運營社團
  4. 搭建青年政策參與溝通平臺

### 4. 秘書室

- **核心職能**：局內行政管理與綜合服務
- **重點工作**：
  1. 文書檔案管理、出納庶務及財產管理
  2. 資訊系統運維與資料管理
  3. 法制研考與公關宣導
  4. 統籌不屬各科室的其他行政事務

### 5. 社會企業中心

- **核心職能**：青年社會企業發展專項推動
- **重點工作**：
  1. 舉辦桃園社企創業競賽（尤努斯獎）
  2. 提供社會企業創業輔導與資源對接
  3. 主導社企創業補助計劃（租金、裝置補貼等）
  4. 發掘青年創新潛力，擴大社會企業影響力

## 三、轄下各基地與資源中心

青年事務局構建了"實體基地+線上平臺"的立體化資源服務網路，為青年提供全方位支援：

### 1. 青年創業基地群（TYC創新創業基地）

- **核心據點**：青創指揮部、安東青創基地、新明青創基地
- **服務內容**：
  - 空間支援：獨立辦公室、共同工作空間、虛擬進駐服務
  - 能力培育：創業主題課程、數位人才培育、科普教育、創客實作
  - 資源對接：創業交流活動、商機媒合平臺
- **資訊獲取**："桃園青創事"Facebook專頁、青年事務局官網

### 2. 青創資源中心（創業資源網）

- **線上平臺**：桃園TYC創新創業資源網（創業一站式平臺）
- **核心服務**：
  1. 創業諮詢輔導與課程公告
  2. 創業團隊精準媒合
  3. 資金對接與政策解讀
- **服務成果**：累計輔導300+青年創業團隊，媒合商機700+件，協助獲取創業資金超3億元，提供諮詢1700+人次

### 3. 青年志工資源中心

- **服務定位**：志願服務資源整合與能力培育平臺
- **核心服務**：
  1. 志工活動資訊釋出與報名
  2. 志工專業培訓課程
  3. 志願服務時長記錄與認證
  4. 國際志工服務隊組建與支援
- **特色專案**：寒暑假返鄉服務營隊、國際志工交流、大型活動志工招募

### 4. 桃園多元青少年活動空間

| 空間名稱 | 定位 | 服務內容 |
|----------|------|----------|
| 桃園設計庫 | 設計及文創青年交流據點 | 創意交流、作品展示、技能共享 |
| 桃園青年體驗學習園區 | 戶外體驗基地 | 戶外教育、團隊培訓、自然體驗 |
| 北區青少年活動中心–桃漾館 | 綜合活動空間 | 多元主題活動、社團聚會、培訓研討 |
| 多元使用空間 | 閒置空間再利用 | 供青年社團申請短期使用 |
| 校園社團資源中心 | 校社對接平臺 | 器材租借、場地資訊、社群連結 |

### 5. 青年e視界

- **平臺定位**：線上資源整合入口
- **核心功能**：
  1. 串聯局屬各網站（創業資源網、設計庫等）
  2. 整合社群賬號（Facebook、Instagram、LINE等）
  3. 彙集"桃青參一咖"專頁、Youthings影音訊道內容
  4. 釋出活動影音、成果報導及福利資訊

## 四、青年補助與服務計劃

針對青年多元發展需求，推出七大類專項補助與服務方案：

### 1. 青年參與多元文化補助

- 補助物件：設籍桃園15～35歲青年、本市團體、學校等
- 補助額度：每件最高新臺幣2萬元
- 申請要求：活動前20日提出申請
- 適用範圍：藝文表演、文化交流等促進青年凝聚力的活動

### 2. 青年參與國際事務補助

- 補助物件：設籍/就學/就業於桃園的15～35歲青年
- 補助範圍：出國參與競賽、展演、大型會議的機票、食宿等費用
- 補助額度：每案最高2萬元
- 申請要求：出國前20日提交計劃書

### 3. 青年社會參與補助

- 補助物件：本市立案團體、學校單位或6人以上青年團隊（15～35歲）
- 補助範圍：公共服務、偏鄉關懷、社群營造等專案
- 補助額度：最高2萬元

### 4. 青年投入永續發展行動計劃

- 招募組別：創新計劃組、領航計劃組
- 支援內容：計劃補助、專業輔導、成果發表機會
- 申請條件：針對在地社群永續議題提出中長服務方案的青年團隊

### 5. 青年國際志工服務隊補助

- 申請條件：6人以上團隊，成員需設籍/就學/就業於桃園
- 補助額度：一般方案最高10萬元；核定志工運用單位最高30萬元

### 6. 社會企業創業補助

- 補助型別：
  - 租金補助：每月最高8,000元，最長2年
  - 裝置補助：最高20萬元（不含消耗品）
- 適用範圍：聚焦弱勢關懷、長照、在地創生、環保、公平貿易等領域的社企

### 7. 青年職涯發展及創業活動補助

- 補助物件：學校、研究機構、法人團體、創業基地團隊
- 補助型別：教育課程、工作坊、論壇、媒合活動、創業競賽等

## 五、常見問答全集

### （一）創業與青創基地相關

1. **問**：青年局的青創基地有辦理相關課程嗎？
   **答**：桃園市政府青年事務局設有青年創業基地（青創指揮部、安東青創基地、新明青創基地），不定時舉辦創業主題式課程、數位人才培育課程、科普教育課程、創客實作課程、創新創業交流聚會等，涵蓋創業核心技能、數位工具應用、微型創業實務。活動資訊可至"桃園青創事"粉絲專頁或青年事務局官網查詢。
   **上版日期**：114-09-30

2. **問**：我剛畢業，有和大學同學共同創立一個青創團隊，想進駐青創基地，請問該如何加入呢？
   **答**：青年局所屬基地（青創指揮部、安東基地）每半年辦理一次進駐審查，招募公告請見"TYC創業資源網"（https://tyc.tycg.gov.tw/）或"青創事"Facebook專頁。
   **上版日期**：114-08-07

3. **問**：我是剛畢業出社會的青年，請問你們是否有創業資源能運用？（如創業基地等）
   **答**：青年事務局提供三大核心資源：①青創指揮部（中壢區，電話03-4279796）；②安東青年創業基地（桃園區，電話03-3355530）；③青創資源中心（創業諮詢與課程，網址http://tyc.tycg.gov.tw/，電話03-4220908）。
   **上版日期**：114-08-07

4. **問**：青創指揮部有什麼樣的設施與裝置呢？該如何申請使用呢？
   **答**：青創指揮部設有獨立辦公室、共同工作空間、會議室、多功能教室與創客空間（含雷射切割機、3D列印機等裝置），可提供一般市民租借。租借申請請至桃園創新創業資源網（https://tyc.tycg.gov.tw/）辦理。
   **上版日期**：114-08-11

5. **問**：我剛創業，目前有資金壓力，請問是否有資金的協助呢？
   **答**：青年可申請"青年創業及中小企業信用保證融資貸款專案""地方型SBIR計劃"等方案，另可至青創資源中心（地址：桃園市中壢區環北路390號3樓，電話(03)422-0908）諮詢創業輔導。
   **上版日期**：114-09-05

### （二）補助申請相關

1. **問**：我是○○團體負責人，想舉辦創業活動，請問是否有補助可以申請呢？
   **答**：可依"青年職涯發展及創業活動補助作業要點"申請，補助物件包括各級學校及研究機構、合法立案的團體/法人/工會/合作社、青年局認定的創業基地進駐團隊；補助活動型別含教育課程、論壇、提案競賽、媒合展覽等。洽詢電話：03-4225205分機5010（王小姐）。
   **上版日期**：114-09-30

2. **問**：我想出國參加比賽、展演或大型國際會議，請問青年事務局有提供相關補助嗎？
   **答**：可依"桃園市政府補助青年參與國際事務作業要點"申請，補助物件為設籍或就學於桃園市的15–35歲青年，最高補助金額可達20萬元。
   **上版日期**：114-08-20

3. **問**：我想舉辦○○○表演活動，請問青年事務局有提供補助嗎？
   **答**：可依"桃園市政府促進青年參與多元文化補助作業要點"申請，鼓勵青年舉辦多元文化、表演藝術、文創設計等活動，最高補助額可達10萬元。
   **上版日期**：114-08-20

4. **問**：我是○○高中學生，想和同學8人於暑假期間，到偏遠地區做關懷服務，請問有相關補助能提供申請嗎？
   **答**：可依"桃園市政府促進青年社會參與補助作業要點"申請，補助專案包括公共議題活動、國內志工服務、國際志工服務等。聯絡：許先生03-422-5205 #6005。
   **上版日期**：114-08-20

### （三）社會參與與職涯相關

1. **問**：我們是一群有服務熱忱的在地青年，想讓桃園更好，請問能協助我們嗎？
   **答**：符合"青年投入永續發展行動計劃"條件可申請參與，該計劃鼓勵15–45歲青年組隊推動地方創生、社群營造、文化儲存等行動，可追蹤"青年事務局"與"桃青參一咖"Facebook專頁掌握徵件資訊。
   **上版日期**：114-08-01

2. **問**：我是○○大學學生，和同學想利用寒暑假期間參加志工服務，請問有相關訊息嗎？
   **答**：可洽"青年志工資源中心"，提供志工培訓、時數證明與活動補助。聯絡電話：03-4225205分機6004、6026；Email：taoyuanyvc@gmail.com。
   **上版日期**：114-08-01

3. **問**：是否有針對青年的職場體驗計劃？
   **答**：青年局推動"桃園青年職場體驗計劃"，邀15–30歲青年參與全臺優質企業職場體驗，瞭解產業趨勢、強化職能競爭力。更多資訊可搜尋"桃園市政府青年事務局"Facebook粉絲專頁。
   **上版日期**：114-08-07

## 六、相關法規與行政規則

### 1. 桃園市政府青年事務局組織規程（摘要）

### 2. 青年事務局補助作業要點

- 涵蓋範圍：多元文化、國際事務、社會參與、職涯及創業活動等補助
- 核心內容：明確申請資格、補助範圍、核銷程式及廉政宣告要求
