## 二、各科室及中心業務職掌
青年事務局依組織規程設立“三科一室”，並下設社會企業中心，形成覆蓋青年發展全領域的業務體系：

### 1. 綜合規劃科
- **核心職能**：青年發展政策研訂與綜合規劃
- **重點工作**：
  1. 編制年度青年政策白皮書
  2. 推動青年多元學習與社團參與
  3. 開展青年設計與藝文發展培育
  4. 組織青年國際交流及體驗學習活動

### 2. 職涯發展科
- **核心職能**：青年就業創業支援與生涯發展引導
- **重點工作**：
  1. 提供青年創業輔導與就業見習服務
  2. 舉辦職涯規劃講座與培訓
  3. 開展創新創業競賽等能力提升活動
  4. 推動青年職場體驗專案

### 3. 公共參與科
- **核心職能**：青年社會參與能力培育與實踐推動
- **重點工作**：
  1. 培育青年公共議題參與意識與能力
  2. 組織志工培訓及志願服務計劃
  3. 輔導青年成立及運營社團
  4. 搭建青年政策參與溝通平臺

### 4. 秘書室
- **核心職能**：局內行政管理與綜合服務
- **重點工作**：
  1. 文書檔案管理、出納庶務及財產管理
  2. 資訊系統運維與資料管理
  3. 法制研考與公關宣導
  4. 統籌不屬各科室的其他行政事務

### 5. 社會企業中心
- **核心職能**：青年社會企業發展專項推動
- **重點工作**：
  1. 舉辦桃園社企創業競賽（尤努斯獎）
  2. 提供社會企業創業輔導與資源對接
  3. 主導社企創業補助計劃（租金、裝置補貼等）
  4. 發掘青年創新潛力，擴大社會企業影響力

## 三、轄下各基地與資源中心
青年事務局構建了“實體基地+線上平臺”的立體化資源服務網路，為青年提供全方位支援：

### 1. 青年創業基地群（TYC創新創業基地）
- **核心據點**：青創指揮部、安東青創基地、新明青創基地
- **服務內容**：
  - 空間支援：獨立辦公室、共同工作空間、虛擬進駐服務
  - 能力培育：創業主題課程、數位人才培育、科普教育、創客實作
  - 資源對接：創業交流活動、商機媒合平臺
- **資訊獲取**：“桃園青創事”Facebook專頁、青年事務局官網

### 2. 青創資源中心（創業資源網）
- **線上平臺**：桃園TYC創新創業資源網（創業一站式平臺）
- **核心服務**：
  1. 創業諮詢輔導與課程公告
  2. 創業團隊精準媒合
  3. 資金對接與政策解讀
- **服務成果**：累計輔導300+青年創業團隊，媒合商機700+件，協助獲取創業資金超3億元，提供諮詢1700+人次

### 3. 青年志工資源中心
- **服務定位**：志願服務資源整合與能力培育平臺
- **核心服務**：
  1. 志工活動資訊釋出與報名
  2. 志工專業培訓課程
  3. 志願服務時長記錄與認證
  4. 國際志工服務隊組建與支援
- **特色專案**：寒暑假返鄉服務營隊、國際志工交流、大型活動志工招募

### 4. 桃園多元青少年活動空間

## 四、青年補助與服務計劃
針對青年多元發展需求，推出七大類專項補助與服務方案：

### 1. 青年參與多元文化補助
- 補助物件：設籍桃園15～35歲青年、本市團體、學校等
- 補助額度：每件最高新臺幣2萬元
- 申請要求：活動前20日提出申請
- 適用範圍：藝文表演、文化交流等促進青年凝聚力的活動

### 2. 青年參與國際事務補助
- 補助物件：設籍/就學/就業於桃園的15～35歲青年
- 補助範圍：出國參與競賽、展演、大型會議的機票、食宿等費用
- 補助額度：每案最高2萬元
- 申請要求：出國前20日提交計劃書

### 3. 青年社會參與補助
- 補助物件：本市立案團體、學校單位或6人以上青年團隊（15～35歲）
- 補助範圍：公共服務、偏鄉關懷、社群營造等專案
- 補助額度：最高2萬元

### 4. 青年投入永續發展行動計劃
- 招募組別：創新計劃組、領航計劃組
- 支援內容：計劃補助、專業輔導、成果發表機會
- 申請條件：針對在地社群永續議題提出中長服務方案的青年團隊

### 5. 青年國際志工服務隊補助
- 申請條件：6人以上團隊，成員需設籍/就學/就業於桃園
- 補助額度：一般方案最高10萬元；核定志工運用單位最高30萬元

### 6. 社會企業創業補助
- 補助型別：
  - 租金補助：每月最高8,000元，最長2年
  - 裝置補助：最高20萬元（不含消耗品）
- 適用範圍：聚焦弱勢關懷、長照、在地創生、環保、公平貿易等領域的社企

### 7. 青年職涯發展及創業活動補助
- 補助物件：學校、研究機構、法人團體、創業基地團隊
- 補助型別：教育課程、工作坊、論壇、媒合活動、創業競賽等

## 五、網站常見問答精選
### 創業相關
1. **問**：剛畢業想創業，有哪些資源？  
   **答**：提供青創基地進駐（青創指揮部、安東基地、新明基地）、創業輔導諮詢、專業課程及貸款利息補貼服務。

2. **問**：如何申請進駐青創基地？  
   **答**：每年分期公開招募，需35歲以下、公司登記於桃園，透過線上報名提交簡報，經資格審查與簡報評選後簽約進駐。

3. **問**：青創基地有開設課程嗎？  
   **答**：有，涵蓋創業技能、數位工具應用、創客實作與創業交流等內容。

4. **問**：剛創業資金不足，可申請哪些協助？  
   **答**：可申請創業貸款利息補貼、青年職涯發展及創業活動補助、社會企業創業補助等。

5. **問**：團體要辦創業活動，有補助嗎？  
   **答**：符合資格的單位可申請青年職涯發展及創業活動補助，覆蓋教育、論壇、競賽等型別。

### 社會參與相關
6. **問**：想參加志工服務，有哪些資源？  
   **答**：青年志工資源中心提供活動資訊、培訓課程，寒暑假期有國內外志工營隊。

7. **問**：在地青年想推社群行動，能獲協助嗎？  
   **答**：可申請青年社會參與補助，或參加“桃青提案所”競賽獲取資金與輔導支援。

## 六、相關法規與行政規則
### 1. 桃園市政府青年事務局組織規程（摘要）
- 制定依據：《桃園市政府組織自治條例》
- 核心架構：設局長1人、副局長1人，下轄三科一室及會計室、人事室
- 會議制度：局務會議由局長召集並擔任主席
- 施行時間：自104年4月1日起施行，112年3月修正最新版本

### 2. 青年事務局補助作業要點
- 涵蓋範圍：多元文化、國際事務、社會參與、職涯及創業活動等補助
- 核心內容：明確申請資格、補助範圍、核銷程式及廉政宣告要求
