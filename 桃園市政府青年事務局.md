## 二、各科室及中心业务职掌
青年事务局依组织规程设立“三科一室”，并下设社会企业中心，形成覆盖青年发展全领域的业务体系：

### 1. 综合规划科
- **核心职能**：青年发展政策研订与综合规划
- **重点工作**：
  1. 编制年度青年政策白皮书
  2. 推动青年多元学习与社团参与
  3. 开展青年设计与艺文发展培育
  4. 组织青年国际交流及体验学习活动

### 2. 职涯发展科
- **核心职能**：青年就业创业支持与生涯发展引导
- **重点工作**：
  1. 提供青年创业辅导与就业见习服务
  2. 举办职涯规划讲座与培训
  3. 开展创新创业竞赛等能力提升活动
  4. 推动青年职场体验项目

### 3. 公共参与科
- **核心职能**：青年社会参与能力培育与实践推动
- **重点工作**：
  1. 培育青年公共议题参与意识与能力
  2. 组织志工培训及志愿服务计划
  3. 辅导青年成立及运营社团
  4. 搭建青年政策参与沟通平台

### 4. 秘书室
- **核心职能**：局内行政管理与综合服务
- **重点工作**：
  1. 文书档案管理、出纳庶务及财产管理
  2. 资讯系统运维与数据管理
  3. 法制研考与公关宣导
  4. 统筹不属各科室的其他行政事务

### 5. 社会企业中心
- **核心职能**：青年社会企业发展专项推动
- **重点工作**：
  1. 举办桃园社企创业竞赛（尤努斯奖）
  2. 提供社会企业创业辅导与资源对接
  3. 主导社企创业补助计划（租金、设备补贴等）
  4. 发掘青年创新潜力，扩大社会企业影响力

## 三、辖下各基地与资源中心
青年事务局构建了“实体基地+线上平台”的立体化资源服务网络，为青年提供全方位支持：

### 1. 青年创业基地群（TYC创新创业基地）
- **核心据点**：青创指挥部、安东青创基地、新明青创基地
- **服务内容**：
  - 空间支持：独立办公室、共同工作空间、虚拟进驻服务
  - 能力培育：创业主题课程、数位人才培育、科普教育、创客实作
  - 资源对接：创业交流活动、商机媒合平台
- **资讯获取**：“桃园青创事”Facebook专页、青年事务局官网

### 2. 青创资源中心（创业资源网）
- **线上平台**：桃园TYC创新创业资源网（创业一站式平台）
- **核心服务**：
  1. 创业咨询辅导与课程公告
  2. 创业团队精准媒合
  3. 资金对接与政策解读
- **服务成果**：累计辅导300+青年创业团队，媒合商机700+件，协助获取创业资金超3亿元，提供咨询1700+人次

### 3. 青年志工资源中心
- **服务定位**：志愿服务资源整合与能力培育平台
- **核心服务**：
  1. 志工活动资讯发布与报名
  2. 志工专业培训课程
  3. 志愿服务时长记录与认证
  4. 国际志工服务队组建与支持
- **特色项目**：寒暑假返乡服务营队、国际志工交流、大型活动志工招募

### 4. 桃园多元青少年活动空间

## 四、青年补助与服务计划
针对青年多元发展需求，推出七大类专项补助与服务方案：

### 1. 青年参与多元文化补助
- 补助对象：设籍桃园15～35岁青年、本市团体、学校等
- 补助额度：每件最高新台币2万元
- 申请要求：活动前20日提出申请
- 适用范围：艺文表演、文化交流等促进青年凝聚力的活动

### 2. 青年参与国际事务补助
- 补助对象：设籍/就学/就业于桃园的15～35岁青年
- 补助范围：出国参与竞赛、展演、大型会议的机票、食宿等费用
- 补助额度：每案最高2万元
- 申请要求：出国前20日提交计划书

### 3. 青年社会参与补助
- 补助对象：本市立案团体、学校单位或6人以上青年团队（15～35岁）
- 补助范围：公共服务、偏乡关怀、社区营造等项目
- 补助额度：最高2万元

### 4. 青年投入永续发展行动计划
- 招募组别：创新计划组、领航计划组
- 支持内容：计划补助、专业辅导、成果发表机会
- 申请条件：针对在地社区永续议题提出中长服务方案的青年团队

### 5. 青年国际志工服务队补助
- 申请条件：6人以上团队，成员需设籍/就学/就业于桃园
- 补助额度：一般方案最高10万元；核定志工运用单位最高30万元

### 6. 社会企业创业补助
- 补助类型：
  - 租金补助：每月最高8,000元，最长2年
  - 设备补助：最高20万元（不含消耗品）
- 适用范围：聚焦弱势关怀、长照、在地创生、环保、公平贸易等领域的社企

### 7. 青年职涯发展及创业活动补助
- 补助对象：学校、研究机构、法人团体、创业基地团队
- 补助类型：教育课程、工作坊、论坛、媒合活动、创业竞赛等

## 五、网站常见问答精选
### 创业相关
1. **问**：刚毕业想创业，有哪些资源？  
   **答**：提供青创基地进驻（青创指挥部、安东基地、新明基地）、创业辅导咨询、专业课程及贷款利息补贴服务。

2. **问**：如何申请进驻青创基地？  
   **答**：每年分期公开招募，需35岁以下、公司登记于桃园，通过线上报名提交简报，经资格审查与简报评选后签约进驻。

3. **问**：青创基地有开设课程吗？  
   **答**：有，涵盖创业技能、数位工具应用、创客实作与创业交流等内容。

4. **问**：刚创业资金不足，可申请哪些协助？  
   **答**：可申请创业贷款利息补贴、青年职涯发展及创业活动补助、社会企业创业补助等。

5. **问**：团体要办创业活动，有补助吗？  
   **答**：符合资格的单位可申请青年职涯发展及创业活动补助，覆盖教育、论坛、竞赛等类型。

### 社会参与相关
6. **问**：想参加志工服务，有哪些资源？  
   **答**：青年志工资源中心提供活动资讯、培训课程，寒暑假期有国内外志工营队。

7. **问**：在地青年想推社区行动，能获协助吗？  
   **答**：可申请青年社会参与补助，或参加“桃青提案所”竞赛获取资金与辅导支持。

## 六、相关法规与行政规则
### 1. 桃园市政府青年事务局组织规程（摘要）
- 制定依据：《桃园市政府组织自治条例》
- 核心架构：设局长1人、副局长1人，下辖三科一室及会计室、人事室
- 会议制度：局务会议由局长召集并担任主席
- 施行时间：自104年4月1日起施行，112年3月修正最新版本

### 2. 青年事务局补助作业要点
- 涵盖范围：多元文化、国际事务、社会参与、职涯及创业活动等补助
- 核心内容：明确申请资格、补助范围、核销程序及廉政声明要求
